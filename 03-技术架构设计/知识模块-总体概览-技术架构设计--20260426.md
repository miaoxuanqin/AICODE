> 更新日期：2026-04-26

## 1. 概述

知识管理模块为住建和综合执法领域提供知识管理、搜索、智能助手服务。

## 5. 模块划分

### 5.1 知识管理模块 (Knowledge)

**功能：**
- 知识 CRUD
- 文件上传解析（PDF/Word → MinIO）
- 关键词搜索（ES）
- 向量索引（Qdrant）
- 收藏、评论

**数据流向：**
```
上传文件 → ParserService → MinIO存储 + ES索引 + Qdrant向量
搜索请求 → ES/Qdrant → 返回知识列表
```

### 5.2 问答模块 (QA)

**功能：**
- 多轮对话会话管理
- RAG 问答（检索 + 生成）
- 热门问题统计
- 问答评价

**RAG 架构：**
```
用户问题 → Embedding → Qdrant向量搜索
              ↓
         ES关键词搜索
              ↓
         知识片段
              ↓
         构建Prompt
              ↓
         MiniMax LLM
              ↓
         生成回答
```

### 5.3 差异化助手配置

| 助手 | Category | 搜索方式 | System Prompt |
|-----|----------|---------|---------------|
| 问答助手 | qa | ES关键词 | 住建领域知识助手 |
| 执法助手 | law_general | ES关键词 | 综合执法领域助手 |
| 工程监管助手 | supervise | 向量语义搜索 | 工程监管领域专家 |

## 6. API 路由

```
/api/v1/knowledge/          # 知识管理
├── GET  /                  # 知识列表
├── POST /                  # 创建知识
├── GET  /{id}             # 知识详情
├── PUT  /{id}             # 更新知识
├── DELETE /{id}           # 删除知识
├── POST /upload            # 文件上传
├── GET  /search            # 搜索知识
├── POST /{id}/favorite     # 收藏
├── POST /{id}/comment      # 评论

/api/v1/qa/                 # 问答助手
├── POST /chat              # 问答对话
├── GET  /sessions          # 会话列表
├── POST /session           # 创建会话
├── GET  /session/{id}      # 会话详情
├── DELETE /session/{id}    # 删除会话
├── GET  /history           # 历史记录
├── GET  /hot-questions     # 热门问题
├── POST /rate              # 评价

/api/v1/auth/               # 认证
├── POST /login
├── POST /logout
├── GET  /me
```

## 7. 数据库模型

### 7.1 知识表 (knowledge)
- id, title, summary, category, source, tags
- file_url, content (ES存储)
- user_id, view_count, created_at

### 7.2 会话表 (qa_session)
- id, user_id, title, category
- is_active, created_at, updated_at

### 7.3 会话消息表 (qa_session_message)
- id, session_id, role, content, created_at
