## 1. 概述

图谱增强 RAG（Graph-RAG）在传统 RAG 基础上引入知识图谱，通过实体和关系建立知识之间的关联，在问答时展示多跳推理路径，增强回答的可解释性和完整性。

**与向量 RAG 的区别**：

| 维度 | 向量 RAG | 图谱增强 RAG |
|------|----------|--------------|
| 检索方式 | 语义相似度 | 图遍历 + 语义 |
| 推理能力 | 单跳 | 多跳 |
| 结果展示 | 知识卡片列表 | 推理路径图 |
| 可解释性 | 低 | 高 |

## 4. 推理流程详解

### Step 1: 实体识别

```python
def extract_entities(self, text: str) -> List[Dict[str, str]]:
    entities = []

    # 规则匹配
    for entity_type, keywords in self.entity_patterns.items():
        for keyword in keywords:
            if keyword in text:
                pattern = rf'{keyword}[^，。！？、\n]{{0,30}}'
                matches = re.findall(pattern, text)
                for match in matches:
                    if match and len(match) > len(keyword):
                        entities.append({"text": match.strip(), "type": entity_type})

    # LLM 辅助提取
    llm_entities = self._extract_entities_with_llm(text)
    # ...

    return entities[:10]
```

### Step 2: 知识检索

```python
# 向量搜索（优先）
query_vector = embedding_service.encode_single(question)
knowledge_ids = search_service.search_vector(query_vector=query_vector, user_id=user_id, limit=6)

# Fallback 到关键词搜索
if vector_search_failed:

```

### Step 3: 关系扩展

```python
def _generate_expand_query(self, original_query: str, search_results: List[Dict]) -> str:
    # 提取搜索结果中的关键主题
    topics = []
    for item in search_results[:3]:
        words = re.findall(r'[一-龥]{2,}', item.get("title", ""))
        topics.extend(words[:2])

    if topics:
        return "、".join(topics[:5])
    return ""
```

### Step 4: 图谱构建

```python
def build_graph(self, entities: List[Dict], search_results: List[Dict]) -> Tuple[List[Dict], List[Dict]]:
    nodes = []
    edges = []
    node_id_map = {}

    # 问题节点
    nodes.append({"id": "q1", "label": entities[0]["text"], "type": "question", ...})
    node_id_map["q1"] = 0

    # 实体节点
    for idx, entity in enumerate(entities[:6]):
        node_id = f"n{idx + 1}"
        nodes.append({"id": node_id, "label": entity["text"][:20], "type": entity["type"], ...})
        edges.append({"id": f"e{idx * 2 + 1}", "source": "q1", "target": node_id, "label": "涉及"})

    # 布局计算
    self._layout_nodes(nodes)

    return nodes, edges
```
