# 技术选型

## 技术栈

| 层级 | 技术 | 说明 |
|-----|------|------|
| 前端框架 | Vue3 + Element Plus | 用户界面 |
| 前端状态 | Vue Router + Pinia | 路由和状态管理 |
| 后端框架 | FastAPI | Python 高性能 API |
| 数据库 | MySQL 8.0 | 元数据存储 |
| 搜索引擎 | Elasticsearch 8.13 | 全文检索 |
| 向量库 | Qdrant 1.17 | 语义搜索 |
| 文件存储 | MinIO | S3 兼容对象存储 |
| 缓存 | Redis | 会话缓存 |
| LLM | MiniMax M2.7 | 对话生成 |
| Embedding | text2vec-base-chinese | 文本向量化（WSL部署） |

## 选型理由

### 前端
- **Vue3**：组合式 API，更好的 TypeScript 支持
- **Element Plus**：Vue3 首选 UI 库，组件丰富
- **Pinia**：Vue3 官方推荐状态管理

### 后端
- **FastAPI**：异步高性能，自动 OpenAPI 文档
- **SQLAlchemy**：Python ORM 标准

### 搜索
- **ES**：成熟全文搜索引擎，关键词搜索首选
- **Qdrant**：向量搜索，支持语义检索

### Embedding
- **text2vec-base-chinese**：轻量中文模型，CPU 可运行
- **部署位置**：WSL (Ubuntu 24.04)
