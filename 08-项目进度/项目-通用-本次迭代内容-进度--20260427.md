## 五、向量分片功能开发（2026-04-27 新增）

### 5.1 问题

原实现只截取文档前 1000 字生成单个向量，文档内容较多时丢失大量信息。

### 5.2 解决方案

修改 `search_service.py` 中的 `index_vector` 方法，实现**文档全文分片向量化**：

| 参数 | 值 | 说明 |
|------|-----|------|
| chunk_size | 500 字符 | 每个 chunk 大小 |
| chunk_overlap | 50 字符 | 相邻 chunk 重叠，保持上下文 |
| 向量维度 | 768 | text2vec-base-chinese |

**分片逻辑：**
```python
def _split_into_chunks(self, text: str, chunk_size: int, overlap: int) -> List[str]:
    """将文本分割成重叠的 chunks"""
    chunks = []
    start = 0
    while start < text_len:
        end = start + chunk_size
        chunk = text[start:end]
        # 去除空行
        chunk = '\n'.join([line.strip() for line in chunk.split('\n') if line.strip()])
        if chunk:
            chunks.append(chunk)
        start += chunk_size - overlap  # 滑动窗口
    return chunks
```

### 5.3 Point ID 修复

**问题：** Qdrant point ID 只接受 UUID 或 unsigned integer，不支持 `{uuid}_{index}` 格式

**解决：** 改用 `uuid.uuid4()` 生成独立 UUID 作为 chunk point ID

### 5.4 user_id 类型修复

**问题：** MySQL 存储 user_id 为整数 `1`，但 Qdrant 过滤时字符串 `"1"` 匹配不到

**解决：** 存储时统一转为字符串 `str(user_id)`

## 七、图谱增强问答Bug修复（2026-04-27 晚）

### 7.1 图谱节点悬停乱跳

**问题**：鼠标悬停在图谱节点上时，节点位置跳动

**原因**：
1. `hoveredNode` 存储整个响应式对象，触发 Vue 重渲染
2. CSS `transform: scale(1.1)` 与 SVG `translate()` 变换冲突

**修复**：
- `GraphQAChat.vue`：`hoveredNode` → `hoveredNodeId`
- CSS：移除 `.node-group:hover` 的 `transform: scale(1.1)`

### 7.2 推理步骤详情无法查看

**问题**：点击推理步骤无法展开查看搜索详情

**原因**：Pydantic `ReasoningStep` 模型缺少 `details` 字段

**修复**：
- `graph_service.py`：每个步骤添加 `details: {search_query, knowledge_found}`
- `graph.py`：`ReasoningStep` 添加 `details: Optional[Dict[str, Any]] = {}`
- `GraphQAChat.vue`：添加 `expandedSteps` 和 `toggleStep()` 实现展开功能

### 7.3 图谱问答 API 404

**问题**：调用 `/api/v1/graph/qa` 返回 404

**原因**：Python `__pycache__` 缓存旧字节码

**修复**：
```bash
find . -type d -name __pycache__ -exec rm -rf {} +
taskkill //F //IM python.exe
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```
