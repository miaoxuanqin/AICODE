## 一、模块概述

**模块名称**：智能助手（Intelligent Assistant）

包含四个子模块：
1. **问答助手** (`/qa/chat`) - 通用知识问答，基于 RAG
2. **执法智能助手** (`/assistant/law`) - 执法场景处置推荐
3. **工程监管助手** (`/assistant/supervise`) - 工程质量安全问题定位
4. **图谱增强问答** (`/assistant/graph`) - 基于知识图谱的多跳推理问答

## 三、本次完成内容（2026-04-28）

### 3.1 思考过程展示与取消功能（四个页面统一修复）

**问题**：原界面等待时只显示简单加载图标，用户可以重复发送问题

**修复**：
1. 发送按钮增加 `|| isThinking` 判断，等待时禁止点击
2. 按钮文字变为"等待回答中..."
3. 进度 15%-100% 时显示取消按钮
4. 点击取消后移除消息、停止动画

**各页面特征**：

| 页面 | 主题色 | 思考步骤 |
|------|--------|----------|
| 问答助手 | #409eff 蓝 | 理解问题 → 知识检索 → 答案生成 |
| 执法智能助手 | #f56c6c 红 | 理解问题 → 法规检索 → 案例匹配 → 方案生成 |
| 工程监管助手 | #67c23a 绿 | 问题分析 → 规范检索 → 案例匹配 → 建议生成 |
| 图谱增强问答 | #1a3a6b 深蓝 | 理解问题 → 知识检索 → 多跳推理 → 生成回答 |

### 3.2 图谱增强问答 Neo4j 状态检查

**问题**：Neo4j 未启动时图谱问答无任何提示

**修复**：
- 前端在 mount 时调用 `neo4jStatus` API
- 未连接时在图谱区域显示红色警告提示
