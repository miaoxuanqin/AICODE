### 二、Bug 修复

#### 1. NEO4J_AVAILABLE 标志缺失
- **文件**：`app/services/neo4j_service.py`
- **问题**：`graph.py` 导入 `NEO4J_AVAILABLE` 失败
- **修复**：在 `neo4j_service.py` 中添加 `NEO4J_AVAILABLE = True`

#### 2. get_center_nodes 边查询逻辑问题
- **文件**：`app/services/neo4j_service.py`
- **问题**：
  - Neo4j 5.x 不支持 `size((n)--())` 语法
  - 边查询要求两边节点都在列表中导致返回空
- **修复**：
  ```python
  # 改用
  size([(n)--() | 1])
  # 边查询改为
  WHERE a.name IN $node_names
  ```

#### 3. DataSet 去重问题
- **文件**：`GraphExplorer.vue`
- **问题**：同一节点 ID 出现在多条边中导致 `DataSet add item with id already exists` 错误
- **修复**：渲染前对节点和边进行去重

#### 4. 字体显示问题
- **问题**：深色节点上白色字体看不清
- **修复**：添加 `strokeWidth: 2, strokeColor: '#000000'` 黑色描边

#### 5. 模板语法错误
- **问题**：`Element is missing end tag` at div.graph-container
- **原因**：el-popover 的闭合标签位置不正确
- **修复**：重构图例部分，使用 legend-item-wrapper 包装每个 popover
