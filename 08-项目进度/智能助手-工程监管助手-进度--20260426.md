### 问题类型(预设)
- 混凝土强度不足
- 钢筋配置不规范
- 防水工程缺陷
- 脚手架安全隐患
- 施工缝处理不当
- 材料以次充好

## 技术方案

### 会话隔离
- category='supervise' 区分执法助手(law_general)和问答助手(qa)
- 不同助手有独立的会话历史

### RAG 流程
```
用户问题 → ES 搜索 → 知识检索 → 构建 Prompt → MiniMax LLM → 生成回答
```

### 前端实现
- initSession：获取或创建 supervise 会话
- loadSessionHistory：加载历史消息
- loadRelatedKnowledge：动态加载关联规范
- handleSend：发送消息并更新会话
