## 模块概述

智能问答系统，基于 RAG 架构（检索增强生成），通过 ES 关键词搜索 + MiniMax LLM 生成回答，支持多轮对话。

## 技术方案

### 多轮对话流程
```
用户提问 → 关联 Session → 获取历史消息（最近5轮）→ ES 搜索 → 构建 Prompt（含历史）
→ MiniMax LLM → 返回回答 → 保存到 Session 消息 → 更新会话标题
```

### 会话隔离方案
- QASession 表新增 category 字段
- category='qa' → 问答助手会话
- category='law_general' → 执法助手会话
- QAChat.vue 加载会话时过滤掉 category='law_general'

### Markdown 渲染方案
- 使用 marked 库渲染 LLM 返回的 Markdown 内容
- 配置 gfm: true（GitHub 风格）和 breaks: true（换行转 <br>）
