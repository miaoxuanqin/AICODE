## 5. 常见问题与解决方案

### 5.1 语义搜索返回 0 条

**排查步骤：**

1. 检查 Qdrant 中的 user_id 类型
```python
# 存储的是字符串 "1" 还是整数 1？
hit.payload.get('user_id')  # 应该是字符串 "1"
```

2. 检查搜索代码中的类型转换
```python
# 正确：直接使用字符串
must_conditions = [FieldCondition(key="user_id", match=MatchValue(value=user_id))]
```

3. 检查 Qdrant Collection 是否存在
```python
client = QdrantClient(url='http://172.20.36.91:6333')
collections = client.get_collections()
print([c.name for c in collections.collections])  # 应该是 ['knowledge']
```

### 5.2 向量维度不匹配

**错误信息：**
```
Vector dimension error: expected dim: 1536, got 768
```

**原因：** Collection 创建时使用了错误的维度

**解决方案：** 重建 Collection
```python
# 删除旧 collection
client.delete_collection('knowledge')

# 重新索引
from app.services.search_service import search_service
from app.models.knowledge import Knowledge
from app.database import SessionLocal

db = SessionLocal()
items = db.query(Knowledge).filter(Knowledge.status == 'published').all()
for item in items:
    search_service.index_vector(item.id, item.title, item.content, item.category, str(item.user_id))
db.close()
```

### 5.3 所有搜索返回相同结果

**可能原因：**
- Embedding 服务返回假数据/相同向量
- Qdrant 中存储的向量都相同

**排查：**
```python
# 检查 Embedding 返回的向量
vec1 = embedding_service.encode_single("文本1")
vec2 = embedding_service.encode_single("文本2")
print(vec1 == vec2)  # 应该是 False
print(vec1[:5])  # 不应该都是 0
```

### 5.4 搜索结果不准确

**可能原因：**
- 知识库内容太少（只有几条数据）
- Embedding 模型效果差
- 向量索引未建立

**解决方案：**
1. 添加更多样例数据
2. 重建向量索引
3. 调整 search_size 参数

## 6. 重建向量索引

### 6.1 手动单条重建

```python
from app.services.search_service import search_service
from app.models.knowledge import Knowledge
from app.database import SessionLocal

db = SessionLocal()
k = db.query(Knowledge).first()

# 获取内容
es_doc = search_service.get_by_id(k.id)
content = es_doc.get('content', '') if es_doc else ''

# 重建索引
search_service.index_vector(
    knowledge_id=k.id,
    title=k.title,
    content=content,
    category=k.category,
    user_id=str(k.user_id)
)

db.close()
```

### 6.2 批量重建

```python
from app.services.search_service import search_service
from app.models.knowledge import Knowledge
from app.database import SessionLocal

db = SessionLocal()
items = db.query(Knowledge).filter(Knowledge.status == 'published').all()

for item in items:
    es_doc = search_service.get_by_id(item.id)
    content = es_doc.get('content', '') if es_doc else ''
    search_service.index_vector(item.id, item.title, content, item.category, str(item.user_id))
    print(f"Indexed: {item.title}")

db.close()
```
