### 3.3 测试向量相似度

```python
from app.services.embedding_service import embedding_service
from qdrant_client import QdrantClient

client = QdrantClient(url='http://172.20.36.91:6333')

# 获取文本向量
text1_vec = embedding_service.encode_single("高处作业安全管理")
text2_vec = embedding_service.encode_single("建筑施工安全")

# 计算余弦相似度
def cosine_sim(a, b):
    dot = sum(x*y for x,y in zip(a,b))
    norm_a = sum(x*x for x in a)**0.5
    norm_b = sum(x*x for x in b)**0.5
    return dot / (norm_a * norm_b)

print(f"相似度: {cosine_sim(text1_vec, text2_vec)}")
```

## 4. 常见问题与解决方案

### 4.1 语义搜索返回 0 条

**排查步骤：**

1. 检查 Qdrant 中的 user_id 类型
```python
# 存储的是字符串 "1" 还是整数 1？
hit.payload.get('user_id')  # 应该是字符串 "1"
```

2. 检查搜索代码中的类型转换
```python
# 错误：转成了整数
user_id_int = int(user_id)  # ❌

# 正确：直接使用字符串
must_conditions = [FieldCondition(key="user_id", match=MatchValue(value=user_id))]
```

### 4.2 向量维度不匹配

**错误信息：**
```
Vector dimension error: expected dim: 1536, got 768
```

**原因：** Collection 创建时使用了错误的维度

**解决方案：** 重建 Collection
```bash
python -m app.scripts.rebuild_vector_index --recreate
```

### 4.3 所有搜索返回相同结果

**可能原因：**
- Embedding 服务返回假数据/相同向量
- Qdrant 中存储的向量都相同

**排查：**
```python
# 检查 Embedding 返回的向量
vec1 = embedding_service.encode_single("文本1")
vec2 = embedding_service.encode_single("文本2")
print(vec1 == vec2)  # 应该是 False

# 检查存储的向量
results = client.query_points(..., with_vectors=True)
print(results.points[0].vector[:5])  # 不应该都是 0
```

### 4.4 搜索结果不准确

**可能原因：**
- 知识库内容太少（只有几条数据）
- Embedding 模型效果差
- 向量索引未建立

**解决方案：**
1. 添加更多样例数据
2. 重建向量索引
3. 调整 search_size 参数

### 5.2 强制重建（删除旧 collection）
```bash
cd code/backend

```

### 5.3 手动单条重建
```python
from app.services.search_service import search_service
from app.models.knowledge import Knowledge
from app.database import SessionLocal

db = SessionLocal()
k = db.query(Knowledge).first()

# 获取内容
es_doc = search_service.get_by_id(k.id)
content = es_doc.get('content', '') if es_doc else ''

# 重建索引
search_service.index_vector(
    knowledge_id=k.id,
    title=k.title,
    content=content,
    category=k.category,
    user_id=str(k.user_id)
)

db.close()
```
