# 知识搜索-语义搜索功能-详细设计

> 更新日期：2026-04-26

## 1. 功能概述

语义搜索功能允许用户通过自然语言描述查找相关知识，无需精确匹配关键词。基于向量相似度计算，返回语义相关的知识条目。

## 2. 用户场景

| 场景 | 搜索词 | 期望找到 | 关键词搜索结果 |
|-----|-------|---------|--------------|
| 工程完工 | "完工" | 竣工验收相关 | 可能不匹配 |
| 法律惩罚 | "处罚" | 违规分包查处案例 | 直接匹配 |
| 高空作业 | "高处作业" | 安全施工规定 | 直接匹配 |
| 违建描述 | "小区有人私搭乱建" | 违法建设强制拆除 | 可能不匹配 |

## 3. 搜索模式

### 3.1 关键词搜索 (keyword)

**实现方式：** ES multi_match 全文搜索

**权重配置：**
- title: 3x
- summary: 2x
- content: 1x

**适用场景：** 已知精确术语，如法规名称、标准编号

### 3.2 语义搜索 (vector)

**实现方式：** Embedding → Qdrant 向量搜索 → ES 详情获取

**流程：**
1. 用户输入自然语言查询
2. Embedding 服务将查询转换为 768 维向量
3. Qdrant 执行余弦相似度搜索，返回 Top-N 知识 ID
4. 根据 ID 从 ES 获取知识详情
5. 返回给前端展示

**适用场景：** 模糊描述、口语化查询、语义理解需求

### 3.3 混合搜索 (hybrid)

**实现方式：** 向量搜索 + 关键词搜索 结果合并

**流程：**
1. 并行执行向量搜索和关键词搜索
2. 向量结果优先，关键词结果补充
3. 去重后返回

**适用场景：** 既想语义理解，又不想遗漏精确匹配

## 4. API 接口

### 4.1 搜索知识

```
GET /api/v1/knowledge/search
```

**请求参数：**
| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| q | string | 是 | 搜索关键词 |
| search_type | string | 否 | keyword/vector/hybrid，默认 keyword |
| category | string | 否 | 知识分类 (law/tech/case/policy) |
| page | int | 否 | 页码，默认 1 |
| page_size | int | 否 | 每页条数，默认 20 |

**响应示例：**
```json
{
  "items": [
```

## 7. 关键技术点

### 7.1 Embedding 服务调用

```python
def encode_single(self, text: str) -> List[float]:
    """单个文本转为向量"""
    resp = self.client.post(
        f"{self.LOCAL_URL}/embed",
        json=[text]
    )
    return resp.json()["embeddings"][0]
```

### 7.2 Qdrant 向量搜索

```python
results = self.qdrant.query_points(
    collection_name="knowledge",
    query=query_vector,  # 768维向量
    query_filter=search_filter,
    limit=limit
)
return [hit.id for hit in results.points]
```

### 7.3 知识ID去重（混合搜索）

```python
seen_ids = set()
merged_items = []

# 向量结果优先
for kid in vector_ids:
    if kid not in seen_ids:
        doc = search_service.get_by_id(kid)
        merged_items.append(doc)
        seen_ids.add(kid)

# 关键词结果补充
for item in keyword_result["items"]:
    if item["id"] not in seen_ids:
        merged_items.append(item)
        seen_ids.add(item["id"])
```

## 8. 限制与注意事项

| 项目 | 说明 |
|-----|------|
| 向量维度 | 768维 (text2vec-base-chinese) |
| user_id 过滤 | Qdrant 存储为字符串，搜索时使用字符串 |
| 相似度阈值 | 默认返回 Top-5，可调整 limit 参数 |
| Embedding 服务 | WSL 本地部署，地址 http://172.20.36.91:8001 |
