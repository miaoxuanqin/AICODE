# 执法智能助手详细设计

> 最后更新：2026-04-27

## 页面概述
- **菜单路径**：智能助手 > 执法智能助手
- **功能**：基于知识图谱的执法处置方案推荐
- **会话 category**：`law_general`

## 页面结构
```
┌─────────────────────────────────────────────────────────────┐
│  执法智能助手                                                 │
│  基于知识图谱的执法处置方案推荐                                  │
├───────────────────────────────────┬─────────────────────────┤
│                                   │  执法场景模板              │
│                                   ├─────────────────────────┤
│         对话区域                   │  关联知识                 │
```

## 功能详情

### 1. 对话功能
- **消息渲染**：支持 Markdown 格式
- **消息结构**：role, content, time, cards, relatedQuestions
- **知识卡片**：显示检索到的相关知识
- **相关问题推荐**：基于检索结果生成

### 2. 场景模板
预定义执法场景模板，点击自动填入输入框：
- 安全隐患检查
- 违规施工处置
- 质量问题查处
- 资质挂靠查处
- 招投标违规

### 3. 关联知识
- 根据最后一条用户消息动态加载
- 显示知识分类标签和标题
- 点击跳转到知识详情页

### 4. 最近案例
- 复用会话历史中的用户问题
- 显示处理状态标签

## API 接口

### 会话管理
| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取会话列表 | GET | /api/v1/qa/sessions | 过滤 category='law_general' |
| 创建会话 | POST | /api/v1/qa/session | category='law_general' |
| 获取会话详情 | GET | /api/v1/qa/session/{id} | 含消息历史 |
| 清空会话消息 | DELETE | /api/v1/qa/session/{id}/messages |

### 问答
| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 问答对话 | POST | /api/v1/qa/chat | 返回 answer, cards |

## 2026-04-27 修复内容

### 1. 卡片字段修复
- **问题**：前端期望 `card.type`，后端返回 `card.category`
- **解决**：后端同时返回 `type` 和 `category` 两个字段

### 2. 会话初始化错误处理
- **问题**：`getSessions` 失败时未创建新会话
- **解决**：catch 块中添加创建新会话逻辑

### 3. 会话加载错误处理
- **问题**：`loadSessionHistory` 失败导致空白页面
- **解决**：catch 块中设置 `messages.value = []` 显示空状态

### 4. 时间格式
- **问题**：消息时间只显示时间无日期
- **解决**：后端返回 `YYYY-MM-DD HH:mm` 格式

### 5. 控制台日志清理
- 移除所有 console.log 调试语句
