# 知识管理新界面详细设计

> 创建时间：2026-04-30
> 更新时间：2026-04-30

## 一、功能概述

基于 `KnowledgeManageNew.html` 原型重构知识管理页面，实现：
- 统计卡片展示（知识总数、ES索引状态、向量搜索状态、图谱状态）
- 列表/网格双视图切换
- 多维度筛选（类型、分类、索引状态）
- 文档上传与解析
- 手动添加文本知识
- 知识状态管理（重建索引、清空索引）
- 文档预览

## 二、接口设计

### 2.1 统计接口

**GET** `/api/v1/knowledge/stats`

**响应**
```json
{
  "total": 156,
  "esIndexed": 142,
  "vectorDone": 128,
  "graphDone": 98
}
```

### 2.2 列表接口扩展

**GET** `/api/v1/knowledge`

响应新增字段：
- `es_indexed`: ES索引状态（indexed/pending/failed/none）
- `vector_indexed`: 向量索引状态（done/pending/failed/none）
- `graph_indexed`: 图谱索引状态（done/pending/failed/none）
- `file_type`: 文件类型（pdf/doc/html）

### 2.3 状态重建接口

**POST** `/api/v1/knowledge/{id}/rebuild/{type}`

**Path Parameters**
- `id`: 知识ID
- `type`: 重建类型 (`es` | `vector` | `graph`)

**响应**: `{"message": "已提交重建任务"}`

### 2.4 状态清空接口

**DELETE** `/api/v1/knowledge/{id}/clear/{type}`

**Path Parameters**
- `id`: 知识ID
- `type`: 清空类型 (`es` | `vector` | `graph`)

**响应**: `{"message": "已清空数据"}`

## 四、前端页面结构

### 4.1 页面布局

```
┌─────────────────────────────────────────────────────────┐
│  页面标题栏                                              │
│  ├── 知识管理 / 管理所有知识内容...                       │
│  └── [添加文本] [上传文档]                              │
│  ├── 统计卡片区 (4列)                                    │
│  │    [知识总数] [ES索引] [向量搜索] [知识图谱]          │
│  ├── 筛选栏                                             │
│  │    [搜索框] [类型▼] [分类▼] [状态标签] [列表|网格]   │
│  ├── 知识列表区                                         │
│  │    (列表视图 或 网格视图)                             │
│  └── 分页器                                             │
└─────────────────────────────────────────────────────────┘
```

### 4.2 组件说明

| 组件 | 说明 |
|-----|------|
| StatsGrid | 4个统计卡片，显示总数、各索引状态及进度条 |
| FilterBar | 搜索框、类型筛选、分类筛选、状态标签、视图切换 |
| KnowledgeList | 列表视图 |
| KnowledgeGrid | 网格视图 |
| UploadDialog | 文件上传弹窗 |
| TextDialog | 文本添加弹窗 |
| PreviewModal | 文档预览弹窗 |
| Pagination | 分页组件 |
