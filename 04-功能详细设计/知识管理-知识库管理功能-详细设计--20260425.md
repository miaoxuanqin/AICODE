# 知识库管理功能详细设计

> 创建时间：2026-04-25
> 更新时间：2026-04-25

## 一、功能目标

提供个人知识库的增删改查能力，用户可以管理自己上传的知识内容，包括查看列表、编辑元数据、删除知识等操作。

## 二、接口说明

### 2.1 知识列表

**GET** `/api/v1/knowledge`

**Query 参数**
| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| page | int | 否 | 页码，默认 1 |
| page_size | int | 否 | 每页条数，默认 20 |
| category | string | 否 | 分类筛选 |
| keyword | string | 否 | 标题关键词搜索 |
| sort | string | 否 | 排序：created_at desc/asc, view_count desc |

**响应**
```json
{
  "code": 200,
  "data": {
    "items": [
      {
        "id": "uuid",
        "title": "建设工程质量管理条例",
        "summary": "为了加强对建设工程质量的管理...",
        "category": "law",
        "category_name": "法律法规",
        "source": "国务院令第279号",
        "tags": ["质量管理", "建设工程"],
        "view_count": 1256,
        "favorite_count": 89,
        "created_at": "2026-04-25T10:00:00Z"
      }
    ],
    "total": 100,
    "page": 1,
    "page_size": 20
  }
}
```

### 2.2 知识详情

**GET** `/api/v1/knowledge/{id}`

**响应**
```json
{
  "code": 200,
  "data": {
    "id": "uuid",
    "title": "建设工程质量管理条例",
    "content": "完整正文内容...",  // 从ES获取
    "summary": "为了加强对建设工程质量的管理...",
    "category": "law",
    "category_name": "法律法规",
    "source": "国务院令第279号",
    "tags": ["质量管理", "建设工程"],
    "file_path": "/knowledge/user_id/uuid.pdf",
    "file_type": "pdf",
    "view_count": 1257,
    "favorite_count": 89,
    "is_favorited": false,
    "created_at": "2026-04-25T10:00:00Z",
    "updated_at": "2026-04-25T10:00:00Z"
  }
}
```

**业务规则**
1. 访问详情时自动 +1 浏览次数
2. **content从ES获取**，MySQL只存元数据
3. 返回 is_favorited 表示当前用户是否已收藏

### 2.3 更新知识

**PUT** `/api/v1/knowledge/{id}`

**请求 Body**
```json
{
  "title": "建设工程质量管理条例（2026修订）",
  "category": "law",
  "source": "国务院令第279号（2026修订）",
  "tags": ["质量管理", "建设工程", "2026修订"]
}
```

**业务规则**
1. 仅可编辑自己的知识
2. 修改后同步更新 ES 索引
3. content需重新上传文件解析修改

### 2.4 删除知识

**DELETE** `/api/v1/knowledge/{id}`

**响应**
```json
{
  "code": 200,
  "message": "删除成功"
}
```

**业务规则**
1. 仅可删除自己的知识
2. 删除时同时清理：
   - MySQL 记录
   - MinIO 文件
   - ES 索引
   - Qdrant 向量（待实现）
3. 删除后不可恢复

## 三、数据权限

| 规则 | 说明 |
|-----|------|
| 用户隔离 | 只能查看/编辑/删除自己的知识 |
| 查询过滤 | 所有查询自动附加 user_id 条件 |
| 管理员 | platform_admin 可查看所有知识（预留） |

## 五、异常处理

| 场景 | 处理方式 |
|-----|---------|
| 知识不存在 | 返回 404 |
| 非所有者访问 | 返回 403 |
| 删除失败 | 返回 500，记录日志，提示重试 |
| ES 同步失败 | 记录错误，异步重试同步 |

## 六、验收标准

1. 用户 A 上传的知识，用户 B 搜索不到
2. 删除知识后，相关文件从 MinIO 删除，ES索引删除
3. 列表分页正常，页码跳转正确
4. 详情页content从ES获取，显示完整正文
