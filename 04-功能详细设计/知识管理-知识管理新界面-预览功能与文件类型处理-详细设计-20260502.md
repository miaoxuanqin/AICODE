# 知识管理-知识管理新界面-预览功能与文件类型处理-详细设计-20260502

> 更新日期：2026-05-02

## 一、功能概述

知识管理新界面的预览功能支持三种类型的知识预览：
1. **PDF文档** - 使用 pdfjs-dist 渲染
2. **Word文档** - 使用 mammoth 转换为 HTML 渲染
3. **文本内容** - 直接显示从 API 获取的内容

### 1.1 预览流程图

```
用户点击知识卡片
       ↓
是否有 file_path?
   ├── 否 → 文本类型知识
   │         ↓
   │    调用 knowledgeApi.get(id) 获取内容
   │         ↓
   │    显示文本预览区域
   │
   └── 是 → 文件类型知识 (PDF/Word)
             ↓
        调用 knowledgeApi.download(id) 获取文件
             ↓
        根据 file_type 决定渲染方式
             ↓
        ┌─── PDF → pdfjs-dist 逐页渲染到 canvas
        ├─── Word → mammoth 转换为 HTML
        └─── 其他 → 提示下载
```

## 二、预览弹窗结构

```html

    </div>

    <!-- Word 预览 -->
    <div v-if="isWordType(previewItem.file_type) && !previewItem.loading"
         class="word-preview text-preview"
```

## 三、预览函数实现

### 3.1 viewFile - 主入口

```javascript
const viewFile = async (item) => {
  const hasFile = item.file_type && item.file_path

  if (hasFile) {
    // 文件类型：PDF/Word
    previewItem.value = { id, title, file_type, url: '', content: '', wordContent: '', loading: true }
    showPreview.value = true

    try {
      const response = await knowledgeApi.download(item.id)
      const rawData = response.data || response
      const blob = new Blob([rawData], { type: 'application/pdf' })

      if (item.file_type === 'pdf') {
        await renderPdfPreview(blob)
      } else if (item.file_type === 'docx') {
        await renderWordPreview(blob)
      }
    } catch (error) {
      ElMessage.error('获取文件失败')
      showPreview.value = false
    }
  } else {
    // 文本类型：无 file_path，直接从 API 获取内容
    previewItem.value = { id, title, file_type, url: '', content: '', wordContent: '', loading: true }
    showPreview.value = true

    try {
      const detail = await knowledgeApi.get(item.id)
      previewItem.value.content = detail.content || ''
    } catch (e) {
      console.error('获取内容失败:', e)
    }
    previewItem.value.loading = false
  }
}
```

### 3.2 renderPdfPreview - PDF 渲染

```javascript
const renderPdfPreview = async (blob) => {
  setupPdfWorker()

  // 等待容器出现（Element Plus 弹窗需要动画时间）
  let container = document.getElementById('pdf-preview-container')
  let waitCount = 0
  while (!container && waitCount < 100) {
    await new Promise(r => setTimeout(r, 100))
    container = document.getElementById('pdf-preview-container')
    waitCount++
  }

  if (!container) return

  try {
    const arrayBuffer = await blob.arrayBuffer()
    const loadingTask = pdfjsLib.getDocument({ data: arrayBuffer })
    const pdf = await loadingTask.promise

    container.innerHTML = ''
    container.style.overflow = 'auto'
    container.style.maxHeight = '70vh'

    for (let i = 1; i <= pdf.numPages; i++) {
      const page = await pdf.getPage(i)
      const scale = 1.5
      const viewport = page.getViewport({ scale })

      const canvas = document.createElement('canvas')
      canvas.style.display = 'block'
      canvas.style.margin = '0 auto 10px'
      canvas.style.boxShadow = '0 2px 8px rgba(0,0,0,0.1)'
      const context = canvas.getContext('2d')
      canvas.height = viewport.height
      canvas.width = viewport.width

      await page.render({ canvasContext: context, viewport }).promise
      container.appendChild(canvas)
    }

    previewItem.value.loading = false
  } catch (error) {
    console.error('PDF 渲染失败:', error)
    previewItem.value.loading = false
  }
}
```

### 3.3 renderWordPreview - Word 渲染

```javascript
const renderWordPreview = async (blob) => {
  try {
    const arrayBuffer = await blob.arrayBuffer()
    const result = await mammoth.convertToHtml({ arrayBuffer })
    previewItem.value.wordContent = result.value
    previewItem.value.loading = false
  } catch (error) {
    console.error('Word 渲染失败:', error)
    previewItem.value.loading = false
  }
}
```

## 四、类型判断辅助函数

```javascript
const isTextType = (fileType) => {
  return !fileType || fileType === 'html' || fileType === 'unknown'
}

const isWordType = (fileType) => {
  return fileType === 'docx' || fileType === 'doc'
}

const isFileType = (fileType) => {
  return fileType === 'pdf' || fileType === 'docx' || fileType === 'doc'
}

const getFileTypeTag = (fileType) => {
  if (fileType === 'pdf') return 'danger'
  return 'primary'
}

const getFileTypeName = (fileType) => {
  if (fileType === 'pdf') return 'PDF文档'
  if (fileType === 'html') return '文本'
  if (fileType === 'docx' || fileType === 'doc') return 'Word文档'
  return '文本'
}
```

## 五、后端接口

### 5.1 获取知识详情

```
GET /api/v1/knowledge/{knowledge_id}
```

返回内容包含 `content` 字段（从 ES 获取的完整内容）。

### 5.2 下载文件

```
GET /api/v1/knowledge/{knowledge_id}/download
```

返回文件 Blob，流式响应。

### 5.3 重建索引

```
POST /api/v1/knowledge/{knowledge_id}/rebuild/{index_type}
```

`index_type` 可选：`es`、`vector`、`graph`

**注意**：图谱重建时从 ES 获取 content（MySQL 不存完整 content）：
```python
content = None
try:
    es_doc = search_service.get_by_id(knowledge_id)
    if es_doc:
        content = es_doc.get("content")
except Exception:
    pass
```

## 八、已解决问题

| 问题 | 解决方案 |
|------|---------|
| PDF 容器找不到 | 添加容器等待循环 |
| PDF.js worker 加载失败 | 使用 `new URL().href` 动态构造 |
| Axios 响应结构错误 | 使用 `response.data || response` |
| 文本类型无预览 | 直接从 API 获取 content |
| 文本类型显示下载按钮 | 添加 `isFileType` 判断 |
