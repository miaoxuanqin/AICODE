## 一、概述

本文档描述知识图谱浏览功能的用户交互优化，包括引导式探索、图例交互增强、搜索图谱联动等功能的设计。

## 二、功能详细设计

### 2.1 引导式探索面板

#### 位置
图谱画布中央，覆盖在空状态或初始加载状态之上

#### 显示条件
```javascript
v-if="showGuidePanel && !loading"
```

#### 界面结构
```
┌─────────────────────────────────────┐
│         📊 探索知识图谱              │
│                                     │
│  输入关键词，或选择下面的示例问题   │
│  开始探索                           │
│                                     │
│  [ 查找海南省住建领域的处罚案例   ] │
│  [ 了解某法规的引用关系           ] │
│  [ 某主体的关联行为有哪些         ] │
│  [ 查找建筑节能相关标准           ] │
│  [ 了解处罚决定的触发条件         ] │
│                                     │
│  💡 提示：您可以搜索实体名称，     │
│      或直接点击图例筛选特定类型    │
│                                     │
│            跳过引导 →               │
└─────────────────────────────────────┘
```

#### 配置数据

### 2.3 搜索+图谱联动

#### 功能
少量搜索结果（≤5个）直接加载到图谱并渲染

#### 实现
```javascript
const handleSearch = async () => {
  // ... 搜索逻辑
  if (res.results.length > 0 && res.results.length <= 5) {
    await loadResultsToGraph(res.results)
    ElMessage.success(`已加载 ${res.results.length} 个相关实体到图谱`)
  }
}

const loadResultsToGraph = async (results) => {
  nodes.value = []
  edges.value = []

  for (const result of results) {
    // 添加搜索结果节点
    newNodes.push({ id: result.id, name: result.name, label: result.label, ... })

    // 获取邻居节点
    const neighbors = await graphApi.neighbors(result.name, 1)
    // 合并节点和边
  }

  nodes.value = newNodes
  edges.value = newEdges

  await nextTick()
  if (network) {
    network.destroy()
    network = null
  }
  renderNetwork()
}
```

### 2.4 智能提示

#### 位置
节点详情面板底部

#### 显示条件
```html
<div v-if="selectedNode && nodeRelations.length > 0" class="node-hint">
```

#### 显示内容
```html
<div class="hint-title">
  <el-icon><InfoFilled /></el-icon>
  关联洞察
</div>
<div class="hint-content">
  <div class="hint-stat">
    该法规关联了 <span class="stat-num">{{ nodeRelations.length }}</span> 个实体
  </div>
  <div class="hint-stat">
    涉及 <span class="stat-num">{{ relatedTypesCount }}</span> 种不同类型
  </div>
</div>
```

#### 计算属性
```javascript
const relatedTypesCount = computed(() => {
  if (!selectedNode.value || !nodeRelations.value.length) return 0
  const types = new Set()
  nodeRelations.value.forEach(rel => {
    if (rel.target_label) types.add(rel.target_label)
    if (rel.source_label) types.add(rel.source_label)
  })
  return types.size
})
```

### 2.5 快捷入口

#### 位置
搜索工具栏下方

#### 按钮配置
```html
<div class="quick-actions">
  <el-button @click="handleQuickAction('cases')">
    <el-icon><Document /></el-icon> 处罚案例
  </el-button>
  <el-button @click="handleQuickAction('laws')">
    <el-icon><Folder /></el-icon> 相关法规
  </el-button>
  <el-button @click="handleQuickAction('random')">
    <el-icon><Refresh /></el-icon> 随便看看
  </el-button>
</div>
```

#### 处理逻辑
```javascript
const handleQuickAction = async (action) => {
  showGuidePanel.value = false
  loading.value = true
  try {
    let results = []
    switch (action) {
      case 'cases':
        const casesRes = await graphApi.searchNodes('', 'Case', 8)
        results = casesRes.results || []
        break
      case 'laws':
        const lawsRes = await graphApi.searchNodes('', 'Law', 8)
        results = lawsRes.results || []
        break
      case 'random':
        const centerRes = await graphApi.centerNodes(8)
        results = centerRes.nodes.map(n => ({ id: n.id, name: n.name, label: n.label }))
        break
    }
    if (results.length > 0) {
      await loadResultsToGraph(results)
    }
  } finally {
    loading.value = false
  }
}
```
