## 一、概述

### 1.1 功能定位

个人中心模块为知识库系统用户提供统一的个人数据管理入口，集中管理用户的收藏知识、历史评论和上传内容。

### 1.2 页面路径

- 路由：`/user/center` 或 `/personal`
- 组件：`code/frontend/src/views/UserCenter.vue`

## 四、API 设计

### 4.1 后端新增接口

#### 4.1.1 获取用户收藏列表

```
GET /api/v1/users/{user_id}/favorites
```

**请求参数**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认1 |
| page_size | int | 否 | 每页条数，默认10 |

**响应**：
```json
{
  "items": [
    {
      "id": "收藏记录ID",
      "knowledge_id": "知识ID",
      "title": "知识标题",
      "summary": "摘要",
      "category": "law",
      "category_name": "法律法规",
      "file_type": "pdf",
      "created_at": "2026-05-10T10:00:00",
      "favorited_at": "2026-05-10T10:00:00"
    }
  ],
  "total": 25,
  "page": 1,
  "page_size": 10
}
```

#### 4.1.2 获取用户评论列表

```
GET /api/v1/users/{user_id}/comments
```

**请求参数**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认1 |
| page_size | int | 否 | 每页条数，默认10 |

**响应**：
```json
{
  "items": [
    {
      "id": "评论ID",
      "knowledge_id": "知识ID",
      "knowledge_title": "知识标题",
      "content": "评论内容",
      "created_at": "2026-05-10T10:00:00"
    }
  ],
  "total": 15,
  "page": 1,
  "page_size": 10
}
```

#### 4.1.3 取消收藏

```
DELETE /api/v1/knowledge/{knowledge_id}/favorite
```

**响应**：
```json

  "message": "已取消收藏"
}
```

#### 4.1.4 删除评论


**响应**：

### 6.1 已有表结构

**user_favorites (已有)**：
| 字段 | 类型 | 说明 |
|------|------|------|
| id | CHAR(36) | 主键 |
| user_id | INT | 用户ID |
| knowledge_id | CHAR(36) | 知识ID |
| created_at | DATETIME | 收藏时间 |

**knowledge_comments (已有)**：
| 字段 | 类型 | 说明 |
|------|------|------|
| id | CHAR(36) | 主键 |
| knowledge_id | CHAR(36) | 知识ID |
| user_id | INT | 用户ID |
| content | TEXT | 评论内容 |
| created_at | DATETIME | 评论时间 |

### 6.2 新增接口SQL

```sql
-- 获取用户收藏列表（带知识详情）
SELECT uf.id, uf.created_at as favorited_at,
       k.id as knowledge_id, k.title, k.summary,
       k.category, k.file_type
FROM user_favorites uf
JOIN knowledge k ON uf.knowledge_id = k.id
WHERE uf.user_id = ?
ORDER BY uf.created_at DESC
LIMIT ? OFFSET ?

-- 获取用户评论列表（带知识标题）
SELECT kc.id, kc.knowledge_id, k.title as knowledge_title,
       kc.content, kc.created_at
FROM knowledge_comments kc
JOIN knowledge k ON kc.knowledge_id = k.id
WHERE kc.user_id = ?
ORDER BY kc.created_at DESC
LIMIT ? OFFSET ?
```
