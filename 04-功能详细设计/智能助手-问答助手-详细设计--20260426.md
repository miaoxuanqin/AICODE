## 一、技术架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   前端页面   │────▶│   FastAPI   │────▶│     LLM     │

└─────────────┘     └──────┬──────┘     └─────────────┘
                           │
                     ┌─────▼─────┐
                     │     ES    │
                     │  关键词搜索│
                     └───────────┘
```

### 核心流程

1. **用户提问** → 前端发送问题到后端
2. **知识检索** → ES 关键词搜索，找到相关知识
3. **构建 Prompt** → 将问题和检索结果组合
4. **LLM 生成** → 调用 MiniMax 模型生成回答
5. **返回结果** → 前端展示答案和知识卡片

### 2.4 LLM 调用

```python
def chat(self, prompt: str):
    client = anthropic.Anthropic(
        base_url=settings.anthropic_base_url,
        api_key=settings.anthropic_auth_token,
    )

    message = client.messages.create(
        model=settings.anthropic_default_haiku_model,
        max_tokens=1024,
        messages=[{"role": "user", "content": prompt}]
    )

    # 解析返回
    # message.content 是列表，包含 ThinkingBlock 和 TextBlock
    for block in message.content:
        if block.type == 'text':
            return block.text

    return ""
```

CREATE TABLE qa_records (
    id VARCHAR(36) PRIMARY KEY,

);

## 七、异常处理

| 场景 | 处理方式 |
|------|----------|
| ES 无搜索结果 | 返回"抱歉，知识库中没有找到相关内容" |
| LLM 调用失败 | 返回"服务暂时不可用，请稍后重试" |
| 网络超时 | 重试 1 次，仍失败返回错误 |
| 无效输入 | 空问题返回提示"请输入问题" |

#### 多轮对话 Prompt
```python
def _build_prompt(self, question, search_results, history_messages):
    # 对话历史（最近 5 轮）
    history_parts = []
    for msg in history_messages:
        role = "用户" if msg['role'] == 'user' else "助手"
        history_parts.append(f"{role}：{msg['content']}")
    history_text = "\n".join(history_parts)

    # 知识内容
    knowledge_parts = []
    for i, item in enumerate(search_results, 1):
        content = item.get('summary', '') or item.get('content', '')[:2000]
        knowledge_parts.append(f"【知识 {i}】\n标题：{item['title']}\n内容：{content}")
    knowledge_text = "\n\n".join(knowledge_parts)

    if history_text:
        return f"""你是一个住建领域知识助手，根据对话历史和知识内容回答用户问题。

【对话历史】
{history_text}

【知识内容】
{knowledge_text}

【当前问题】
{question}

请根据对话历史和知识内容给出准确、专业的回答。"""
    else:
        return f"""你是一个住建领域知识助手，根据以下知识内容回答用户问题。

【知识内容】
{knowledge_text}

【用户问题】
{question}

请根据知识内容给出准确、专业的回答。"""
```
