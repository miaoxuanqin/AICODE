## 一、本次优化概述

### 1.1 优化背景

图谱增强问答上线后，用户反馈等待答案时界面缺乏动态效果，推理过程不透明。本次优化重点提升用户体验。

### 1.2 优化内容

| 优化项 | 说明 |
|--------|------|
| 开关文字优化 | "使用Neo4j" → "图谱推理" |
| 默认图谱推理 | useNeo4j 默认为 true |
| 思考中界面 | 实时展示推理链步骤，替代原有简单进度条 |
| 趣味提示 | 轮换显示图谱相关提示语，减少等待焦虑 |
| 图谱动态效果 | 节点弹入动画、边绘制动画 |
| 动态图谱生成 | 后端无数据时前端动态生成相关节点 |
| 第4步详情增强 | 最终步骤显示进度条和提示，增强等待体验 |
| 节点尺寸优化 | 缩小节点半径避免重叠 |
| 边样式优化 | 边颜色加深，文字背景高亮 |

### 3.2 动画逻辑

```javascript
const startThinkingAnimation = () => {
  let progress = 0
  liveReasoningChain.value = []
  currentTipIndex.value = 0

  // 先添加所有4个步骤
  REASONING_STEPS.forEach((step, idx) => {
    liveReasoningChain.value.push({
      query: step.query,
      result: step.result,
      entities: []
    })
  })
  thinkingCurrent.value = 0

  const timer = setInterval(() => {
    progress += Math.random() * 12 + 4
    if (progress > 100) progress = 100
    thinkingProgress.value = Math.round(progress)

    // 根据进度更新当前步骤
    const totalSteps = REASONING_STEPS.length
    const stepThreshold = 100 / totalSteps
    const stepIndex = Math.min(Math.floor(progress / stepThreshold), totalSteps - 1)
    thinkingCurrent.value = stepIndex

    // 更新每个步骤的实体显示
    liveReasoningChain.value.forEach((item, idx) => {
      if (idx <= stepIndex && idx > 0) {
        item.entities = generateMockEntities(idx)
      }
    })

    // 轮换趣味提示
    if (Math.random() < 0.3) {
      currentTipIndex.value = (currentTipIndex.value + 1) % funTips.length
    }

    if (progress >= 100) {
      clearInterval(timer)
    }
  }, 250)
  return timer
}
```

## 四、图谱动态效果

### 4.1 分步添加动画

```javascript
// 清空图谱
graphData.nodes = []
graphData.edges = []

// 逐步添加节点
let nodeIdx = 0
const addNode = () => {
  if (nodeIdx < finalNodes.length) {
    graphData.nodes.push(finalNodes[nodeIdx])
    nodeIdx++
    setTimeout(addNode, 400)
  } else {
    // 节点添加完，添加边
    let edgeIdx = 0
    const addEdge = () => {
      if (edgeIdx < finalEdges.length) {
        graphData.edges.push(finalEdges[edgeIdx])
        edgeIdx++
        setTimeout(addEdge, 300)
      }
    }
    setTimeout(addEdge, 300)
  }
}
setTimeout(addNode, 300)
```

## 六、动态图谱生成

### 6.1 生成逻辑

```javascript
const generateDynamicGraphData = (question, reasoningChain) => {
  const nodes = []
  const edges = []

  // 问题节点 - 顶部居中
  nodes.push({
    id: 'q1', label: question, type: 'question',
    x: 200, y: 50, description: question, attributes: []
  })

  // 知识节点 - 分散排列
  const positions = [
    { x: 60, y: 150 }, { x: 60, y: 280 }, { x: 200, y: 200 },
    { x: 340, y: 150 }, { x: 340, y: 280 }
  ]
  uniqueKnowledge.forEach((title, idx) => {
    nodes.push({
      id: `n${idx+1}`, label: title, type: types[idx % types.length],
      x: positions[idx].x, y: positions[idx].y, ...
    })
    edges.push({ id: `e${idx+1}`, source: 'q1', target: `n${idx+1}`, label: '涉及' })
  })

  // 实体节点 - 底部一排
  const entityX = [80, 160, 240, 320]
  uniqueEntities.forEach((entity, idx) => {
    nodes.push({ id: `e${idx+10}`, label: entity, type: 'policy', x: entityX[idx], y: 340, ... })
  })

  return { nodes, edges }
}
```
