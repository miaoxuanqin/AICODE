# 知识搜索功能详细设计

> 创建时间：2026-04-25
> 更新时间：2026-04-25

## 一、功能目标

实现多维度的知识搜索能力，支持关键词搜索和语义搜索（向量相似度），支持分类、来源、日期等多条件筛选，返回排序后的结果。

## 二、搜索类型

### 2.1 关键词搜索

基于 Elasticsearch 的倒排索引，实现分词匹配、相关性评分。

**特点**：
- 使用默认分词器
- 按相关度排序

### 2.2 语义搜索（待实现）

基于 Qdrant 向量数据库，实现语义理解搜索。

**特点**：
- 理解查询意图，不依赖字面匹配
- 支持"如何处理..."、"什么情况要..."等问句
- 返回 top-k 最相似结果
- **需接入embedding模型**

## 三、接口说明

### 3.1 聚合搜索

**GET** `/api/v1/knowledge/search`

**Query 参数**
| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| q | string | 是 | 搜索关键词/问句 |
| search_type | string | 否 | keyword/vector/hybrid，默认 hybrid |
| category | string | 否 | 分类筛选 |
| source | string | 否 | 来源筛选 |
| date_from | string | 否 | 开始日期 YYYY-MM-DD |
| date_to | string | 否 | 结束日期 YYYY-MM-DD |
| page | int | 否 | 页码，默认 1 |
| page_size | int | 否 | 每页条数，默认 20 |
| sort | string | 否 | 排序：relevance/date/views，默认 relevance |

**响应**
```json
{
  "code": 200,
  "data": {
    "items": [
      {
        "id": "uuid",
        "title": "建设工程质量管理条例",
        "summary": "为了加强对建设工程质量的管理...",
        "category": "law",
        "category_name": "法律法规",
        "source": "国务院令第279号",
        "tags": ["质量管理", "建设工程"],
        "view_count": 1256,
        "score": 0.95,
        "highlight": {
          "title": "建设工程质量<em>管理</em>条例",
          "content": "...加强<em>建设</em>工程质量<em>管理</em>..."
        }
      }
    ],
    "total": 50,
    "page": 1,
    "page_size": 20,
    "search_type": "keyword"
  }
}
```

### 3.2 搜索建议（自动补全）

**GET** `/api/v1/knowledge/suggest`

**Query 参数**
| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| q | string | 是 | 前缀 |
| limit | int | 否 | 返回数量，默认 10 |

**响应**
```json
{
  "code": 200,
  "data": [
    { "text": "建设工程质量管理", "count": 10 },
    { "text": "建设工程安全生产", "count": 8 },
    { "text": "建筑工程施工许可", "count": 5 }
  ]
}
```

### 3.3 热门知识

**GET** `/api/v1/knowledge/hot`

**Query 参数**
| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| limit | int | 否 | 返回数量，默认 10 |

**响应**
```json

      "title": "建设工程质量管理条例",
```

### 3.4 最新知识

**GET** `/api/v1/knowledge/latest`

**Query 参数**
| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| limit | int | 否 | 返回数量，默认 10 |

## 四、搜索流程

### 4.1 关键词搜索（当前实现）

```
1. 解析查询参数
2. 关键词搜索（ES）
   - 查询 title、content、summary 字段
   - 使用默认分词
   - 返回相关度评分
3. 高亮处理
   - ES 返回 highlight 信息
4. 返回结果
```

### 4.2 Hybrid 搜索（待完善）

```
1. 解析查询参数
2. 关键词搜索（ES）
3. 语义搜索（Qdrant）- 待接入embedding
4. 结果融合
   - RRF (Reciprocal Rank Fusion) 算法合并排名
5. 返回结果
```

## 五、数据权限

| 规则 | 说明 |
|-----|------|
| 用户隔离 | 搜索结果只返回当前用户的知识 |
| 视角过滤 | 可按视角（省/市县）进一步过滤（预留） |

## 七、异常处理

| 场景 | 处理方式 |
|-----|---------|
| ES 不可用 | 返回 503 |
| Qdrant 不可用 | 降级为纯关键词搜索 |
| 无搜索结果 | 返回空列表，提示尝试其他关键词 |
| embedding 服务不可用 | 语义搜索暂不可用 |

## 八、验收标准

1. 搜索"建设工程"，能返回包含该关键词的知识
2. 高亮显示匹配关键词
3. 分类筛选、排序功能正常
4. 搜索响应时间 < 2 秒
