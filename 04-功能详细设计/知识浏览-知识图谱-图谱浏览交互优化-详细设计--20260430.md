## 一、需求背景

用户反馈图谱浏览（GraphExplorer.vue）存在以下体验问题：

1. **关联关系为空时无感知**：点击"抽取自"等系统关系无反应
2. **相关推荐点击无反馈**：点击推荐节点后图谱无变化
3. **关联关系点击无高亮**：点击关系项后目标节点未高亮显示

本次优化旨在提升图谱交互的反馈感，让用户清楚感知每次操作的效果。

---

## 二、问题分析

### 2.1 问题一：关联关系点击无反应

**现象**：某些关系（如"抽取自"EXTRACTED_FROM）指向的节点名称为空，导致点击后无法导航。

**原因**：
- 前端 `handleRelationClick` 直接使用空的目标名称查询
- 后端 `get_node_relations` 返回了指向匿名知识引用节点的关系

**解决方案**：
- 前端：增加空名称检查，提示用户"该关系指向系统引用节点"
- 后端：过滤掉 `connected.name IS NULL` 的关系

### 2.2 问题二：相关推荐点击无反馈

**现象**：点击相关推荐中的项目后，图谱没有明显变化。

**原因**：
- `loadNodeAndFocus` 函数在加载节点后未正确高亮
- 渲染网络后没有触发高亮动画

**解决方案**：
- 增加 `highlightSingleNode` 函数，绿色高亮显示选中的节点
- 在加载节点后延迟 500-600ms 调用高亮
- 相关推荐点击使用独立处理函数 `handleRecommendationClick`

### 2.3 问题三：关联关系点击无高亮

**现象**：点击关联关系列表中的项目，虽然会跳转，但图谱中没有高亮提示。

**解决方案**：
- 调用 `network.focus()` 后同步调用 `highlightSingleNode()`

### 3.1 后端修复

**文件**：`code/backend/app/services/neo4j_service.py`

**修改位置**：`get_node_relations` 方法（第 1018 行附近）

**修改内容**：
```python
# 修改前
query = """
MATCH (center)-[r]-(connected)
WHERE center.name = $node_name
RETURN center, r, connected, labels(center)[0] AS center_label, labels(connected)[0] AS connected_label,
       startNode(r) = center AS is_outgoing
"""

# 修改后
query = """
MATCH (center)-[r]-(connected)
WHERE center.name = $node_name AND connected.name IS NOT NULL
RETURN center, r, connected, labels(center)[0] AS center_label, labels(connected)[0] AS connected_label,
       startNode(r) = center AS is_outgoing
"""
```

**作用**：过滤掉指向匿名知识引用节点（无 name 属性）的关系

```javascript
// 高亮单个节点
const highlightSingleNode = (nodeId) => {
  console.log('highlightSingleNode 被调用, nodeId:', nodeId, 'network:', !!network)
  if (!network) return

  const allNodes = network.body.data.nodes.get()
  console.log('当前网络节点数:', allNodes.length)
  const targetNode = allNodes.find(n => n.id === nodeId)
  console.log('目标节点是否存在:', !!targetNode)

  allNodes.forEach(node => {
    const originalNode = nodes.value.find(n => n.id === node.id)
    const isHighlighted = node.id === nodeId
    network.body.data.nodes.update({
      id: node.id,
      color: {
        background: isHighlighted ? '#67c23a' : (originalNode?.isNew ? '#ff6b35' : getNodeColor(node.type || originalNode?.type)),
        border: isHighlighted ? '#000000' : '#ffffff',
        highlight: { background: '#67c23a', border: '#000000' }
      },
      borderWidth: isHighlighted ? 5 : 1,
      font: { color: isHighlighted ? '#ffffff' : '#ffffff', size: isHighlighted ? 16 : 12, strokeWidth: 3, strokeColor: '#000000' }
    })
  })

  // 3秒后恢复颜色
  setTimeout(() => {
    updateNodeColors()
  }, 3000)
}
```

#### 3.2.2 修改 handleRelationClick

```javascript
// 处理关系点击
const handleRelationClick = (rel) => {
  const targetName = rel.direction === 'outgoing' ? rel.target_name : rel.source_name
  // 跳过空名称的关系（系统引用节点，如抽取自关系指向的知识节点）
  if (!targetName) {
    ElMessage.info('该关系指向系统引用节点，无法导航')
    return
  }
  const targetNode = nodes.value.find(n => n.name === targetName)
  if (targetNode) {
    selectNode(targetNode)
    if (network) {
      network.focus(targetNode.id, { scale: 1.5, animation: true })
      highlightSingleNode(targetNode.id)  // 新增高亮
    }
  } else {
    loadNodeAndFocus(null, targetName)
  }
}
```

#### 3.2.3 新增 handleRecommendationClick

```javascript
// 处理推荐项目点击
const handleRecommendationClick = (rec) => {
  const targetNode = nodes.value.find(n => n.id === rec.id || n.name === rec.name)
  if (targetNode) {
    selectNode(targetNode)
    if (network) {
      network.focus(targetNode.id, { scale: 1.5, animation: true })
      highlightSingleNode(targetNode.id)
    }
  } else {
    loadNodeAndFocus(rec.id, rec.name)
  }
}
```

#### 3.2.4 修改 loadNodeAndFocus

在聚焦到目标节点后增加高亮调用：

```javascript
// 聚焦到目标节点
const targetNode = nodes.value.find(n => n.name === nodeName)

    console.log('调用 highlightSingleNode:', targetNode.id)

```

#### 3.2.5 修改相关推荐过滤

```javascript
// 加载相关推荐
const loadRelatedRecommendations = async () => {
  if (!selectedNode.value) return
  try {
    const res = await graphApi.neighbors(selectedNode.value.name, 2)
    if (res.nodes) {
      // 获取二级邻居作为推荐（过滤掉无名称的节点）
      const recommendations = res.nodes
        .filter(n => n.id !== selectedNode.value.id && n.name)  // 新增 n.name 检查
        .slice(0, 5)
        .map(n => ({ id: n.id, name: n.name, label: n.label }))
      relatedRecommendations.value = recommendations
      if (recommendations.length === 0) {
        ElMessage.info('暂无相关推荐')
      }
    }
  } catch (error) {
    console.error('加载相关推荐失败:', error)
  }
}
```
