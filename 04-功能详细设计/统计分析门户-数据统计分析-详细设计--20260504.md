## 一、功能概述

统计分析门户是为知识库系统设计的全新数据统计和分析页面，通过可视化的图表和卡片展示系统运行状态、知识资产状况和用户活动情况。

### 1.1 主要功能

1. **统计卡片展示**：展示关键业务指标（知识总数、本月新增、索引进度等）
2. **知识增长趋势**：通过面积折线图展示知识增长趋势
3. **分类分布分析**：环形图/玫瑰图展示知识分类占比
4. **索引状态监控**：堆叠柱状图展示各类型索引状态
5. **热门标签分析**：水平柱状图/词云展示热门标签TOP15
6. **知识来源统计**：柱状图展示知识来源TOP10
7. **用户活跃度分析**：热力日历图展示用户活跃时段
8. **快捷操作入口**：一键访问常用功能
9. **实时动态跟踪**：时间线展示最新系统动态
10. **索引进度监控**：实时显示索引任务进度

### 8.3 前端数据加载

```javascript
const loadPortalStats = async () => {
  try {
    loading.value = true
    const params = {}
    if (dateRange.value && dateRange.value.length === 2) {
      params.date_from = dateRange.value[0]
      params.date_to = dateRange.value[1]
    }
    const res = await knowledgeApi.portalStats(params)
    portalData.value = res
    updateStatsCards()
    updateCharts()
  } catch (error) {
    console.error('加载门户数据失败', error)
  } finally {
    loading.value = false
  }
}
```

### 8.4 数据更新机制

- **页面加载时**：自动调用 `loadPortalStats()` 获取初始数据
- **刷新按钮**：调用 `refreshData()` 重新加载
- **日期筛选变化**：调用 `handleDateChange()` 重新加载

## 九、已知问题与解决

### 9.1 MySQL JSON 字段聚合

**问题**：`tags` 是 JSON 类型，直接 GROUP BY 报错

**解决方案**：Python 中手动遍历统计

```python
tag_count_map = {}
tag_query = db.query(Knowledge.tags).filter(
    Knowledge.tags.isnot(None),
    Knowledge.user_id == str(current_user.id) if not is_admin else True
).all()
for row in tag_query:
    if row.tags:
        for tag in row.tags:
            tag_count_map[tag] = tag_count_map.get(tag, 0) + 1
```
