# 知识管理-门户控制台-原型设计-20260430

## 1. 概述

本文档描述住建知识库系统门户控制台的原型设计，作为系统的"门面"给用户第一眼的感受，需要：大气、专业、有权威感。

## 2. 原型文件

| 文件 | 说明 |
|------|------|
| `code/frontend/src/views/Portal.html` | 门户控制台原型（独立HTML） |

## 3. 设计定位

作为整个系统的入口门户，面向管理员和普通用户，展示系统核心数据和快捷入口。

## 4. 功能模块

### 4.1 快捷操作

4个快捷按钮：
- 上传知识
- 智能问答
- 知识图谱
- 文档库

### 4.2 统计卡片

| 卡片 | 内容 |
|------|------|
| 知识总数 | 总数、文档数、文本数、环比增长 |
| 全文检索 | 已索引数、占总数百分比 |
| 语义搜索 | 已处理数、待处理数、百分比 |
| 知识图谱 | 已构建数、待构建数、百分比 |

### 4.3 功能模块入口

4大功能卡片：
- 知识管理
- 智能问答
- 知识图谱
- 文档库

每个卡片包含：图标、名称、描述、hover箭头动画

### 4.4 索引进度

进度条展示：
- 全文检索进度（绿色渐变）
- 语义搜索进度（蓝色渐变）
- 知识图谱进度（紫色渐变）

### 4.5 分类分布

4个分类的统计：
- 法律法规（红色）
- 技术标准（绿色）
- 执法案例（橙色）
- 政策文件（蓝色）

### 4.6 最新知识列表

显示5条最新添加的知识，包含：
- 类型图标
- 标题
- 来源
- 日期
- 分类标签

### 4.7 热门标签云

显示8个热门标签，可点击跳转搜索

### 4.8 存储统计

- PDF文件数量
- Word文件数量
- 存储空间使用率

## 5. 视觉设计

### 5.1 配色方案

沿用政务蓝主题：
- 主色：`#1e4d7b`（深蓝）
- 辅助：`#2d5a8a`（中蓝）
- 点缀：`#daa520`（金色）
- 背景：`#f0f4f8`（浅灰蓝）

### 5.2 动效设计

- 页面加载：卡片依次淡入上浮（stagger 动画）
- hover：卡片轻微上浮 + 阴影加深
- 按钮：点击缩放反馈

## 6. 统计数据项汇总

| 类别 | 统计项 |
|------|--------|
| **基础统计** | 总数、文档数、文本数、本月新增 |
| **索引状态** | ES索引率、向量化率、图谱覆盖率 |
| **分类统计** | 法律法规/技术标准/执法案例/政策文件数量 |
| **文件统计** | PDF数、Word数、总存储大小 |
| **用户活动** | 问答次数、活跃用户数 |
| **热门标签** | 标签名称 + 使用次数 |

- 对接后端 API 获取真实数据
