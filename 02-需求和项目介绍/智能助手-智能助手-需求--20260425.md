# 智能助手功能点

> 创建时间：2026-04-25

## 一、功能概述

智能助手分为执法智能助手和工程监管智能助手，通过对存量知识的挖掘和知识图谱技术的结合，达到对综合执法案件、工程质量安全监管问题进行图谱可视化展示、分析与推荐。

## 二、执法智能助手

### 2.1 功能清单

| 功能点 | 描述 | 优先级 |
|-------|------|--------|
| 场景描述输入 | 输入执法场景描述 | P0 |
| 处置方案推荐 | 基于知识图谱推荐处置方案 | P0 |
| 法律法规引用 | 引用相关法律法规依据 | P0 |
| 类似案例参考 | 提供类似案例参考 | P1 |
| 场景模板 | 提供常见执法场景模板 | P1 |

### 2.2 执法场景模板

| 场景类型 | 说明 |
|---------|------|
| 工地安全违规 | 施工安全隐患类 |
| 违法建设 | 未批先建类 |
| 施工许可违规 | 无证施工类 |
| 质量安全隐患 | 工程质量类 |
| 扬尘污染 | 环保类 |
| 综合执法通用 | 综合类 |

### 2.3 输出内容

- 处置步骤建议
- 适用的法律法规条款
- 处罚依据和标准
- 类似案例处置结果

## 三、工程监管智能助手

### 3.1 功能清单

| 功能点 | 描述 | 优先级 |
|-------|------|--------|
| 问题描述输入 | 输入现场发现的问题 | P0 |
| 问题类型识别 | 识别问题类型 | P0 |
| 技术标准引用 | 引用相关技术标准 | P0 |
| 处置建议 | 提供整改处置建议 | P0 |
| 规范查询 | 查询相关规范标准 | P1 |

### 3.2 常见问题类型

| 问题类型 | 说明 |
|---------|------|
| 混凝土强度不足 | 砼强度问题 |
| 钢筋配置不规范 | 钢筋工程问题 |
| 防水工程缺陷 | 防水问题 |
| 脚手架安全隐患 | 安全设施问题 |
| 施工缝处理不当 | 施工工艺问题 |
| 材料以次充好 | 材料问题 |

## 四、页面原型

### 4.1 执法智能助手
- 路径：`code/frontend/src/views/assistant/LawAssistant.vue`
- 左侧：对话区域
- 右侧：场景模板 + 关联知识 + 最近案例

### 4.2 工程监管助手
- 路径：`code/frontend/src/views/assistant/SuperviseAssistant.vue`
- 左侧：对话区域
- 右侧：问题类型 + 规范标准 + 监管记录

## 五、技术实现

### 5.1 对话流程

```
用户输入 → 意图识别 → 实体提取 → 知识图谱查询 → 方案生成 → 结果展示
```

### 5.2 数据来源

- 知识库内容
- 知识图谱实体
- 历史执法案例
- 法律法规库
