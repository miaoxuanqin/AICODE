# 需求理解总览

> 创建时间：2026-04-25

## 一、系统定位

海南省住建知识库是行业知识图谱模块的核心组成部分，基于省级建设全省住建和综合执法领域知识体系的构建和服务标准规范，为业务人员提供一站式知识检索、智能辅助决策服务。

## 二、系统架构

### 2.1 模块划分

| 模块名称 | 功能定位 |
|---------|---------|
| 知识搜索门户 | 一站式知识检索入口 |
| 执法智能助手 | 综合执法案件处置推荐 |
| 工程监管助手 | 质量安全问题定位与处置 |
| 问答助手 | 常见问题智能问答 |
| 系统管理 | 用户、角色、权限管理 |

### 2.2 技术架构

- **后端**：FastAPI (Python 3.10+)
- **前端**：Vue 3 + Element Plus
- **数据库**：MySQL 8.0
- **缓存**：Redis
- **搜索引擎**：Elasticsearch 8.13
- **向量库**：Qdrant
- **文件存储**：MinIO (S3兼容)
- **消息队列**：Celery + Redis

## 三、业务流程

```
用户登录 → 视角选择 → 知识搜索/智能助手/问答 → 知识详情 → 评论/收藏/评价
                                      ↓
                              系统管理（管理员）
```

## 四、数据权限体系

### 4.1 组织架构

```
海南省住房和城乡建设厅（省级）
├── 海口市住房和城乡建设局（市县级）
├── 三亚市住房和城乡建设局（市县级）
└── 其他市县...
```

### 4.2 角色体系

| 角色标识 | 角色名称 | 权限范围 |
|---------|---------|---------|
| platform_admin | 平台管理员 | 系统最高权限 |
| provincial_admin | 省厅管理员 | 全省知识管理 |
| city_admin | 市县管理员 | 本地知识管理 |
| business_user | 业务用户 | 基础业务操作 |
| viewer | 访客 | 只读权限 |

### 4.3 权限控制维度

1. **模块权限**：功能操作权限（搜索/配置/管理等）
2. **数据权限**：知识访问范围（按组织/分类/密级）
3. **组织范围**：地域限制（省级/市县级）

## 五、非功能性需求

- 响应时间：搜索结果 < 2秒
- 并发支持：支持500+并发用户
- 数据更新：增量采集+全量采集
- 安全要求：JWT认证、权限细粒度控制
