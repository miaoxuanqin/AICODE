# 知识管理模块功能点

> 创建时间：2026-04-25

## 一、功能概述

知识管理模块为用户提供独立的知识库管理能力，支持文件上传自动解析入库，形成个人或组织的知识资产。支持关键词搜索和语义搜索，实现知识的快速检索与复用。

## 二、核心特性

### 2.1 多租户隔离

- 每个用户拥有独立的知识库
- 知识数据按 user_id 隔离
- 用户只能搜索和管理自己的知识
- 支持知识分享（可选）

### 2.2 文件上传解析

| 特性 | 说明 |
|-----|------|
| 支持格式 | Word (.doc/.docx)、PDF |
| 存储方式 | 原始文件存储到 MinIO |
| 解析方式 | 自动提取标题、文本内容、摘要 |
| 入库方式 | 无需审核，解析后直接入库 |

### 2.3 知识类型

| 类型 | 说明 |
|-----|------|
| 法律法规 | 国家法规、省级法规、地方条例 |
| 技术标准 | 行业标准、技术规范、操作规程 |
| 执法案例 | 行政处罚案例、整改案例、事故案例 |
| 政策文件 | 通知、办法、决定、意见 |

## 四、数据模型

### 4.1 知识表 (knowledge)

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | UUID | 主键 |
| title | VARCHAR(500) | 知识标题 |
| content | **MEDIUMTEXT** | **正文存ES，MySQL允许NULL** |
| summary | TEXT | 摘要 |
| category | VARCHAR(50) | 分类：law/tech/case/policy |
| source | VARCHAR(500) | 来源/发布部门 |
| tags | JSON | 标签数组 |
| user_id | UUID | 所属用户 |
| file_path | VARCHAR(1000) | MinIO 存储路径 |
| file_type | VARCHAR(20) | 文件类型：docx/pdf |
| view_count | INT | 浏览次数 |
| favorite_count | INT | 收藏次数 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 4.2 用户收藏表 (user_favorite)

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | UUID | 主键 |
| user_id | UUID | 用户ID |
| knowledge_id | UUID | 知识ID |
| created_at | DATETIME | 收藏时间 |

### 4.3 知识评论表 (knowledge_comment)

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | UUID | 主键 |
| knowledge_id | UUID | 知识ID |
| user_id | UUID | 评论用户 |
| content | TEXT | 评论内容 |
| created_at | DATETIME | 评论时间 |

## 五、技术实现

### 5.1 技术栈

| 组件 | 用途 |
|-----|------|
| FastAPI | 后端 API 框架 |
| SQLAlchemy | ORM |
| MySQL | 关系数据库 |
| MinIO | 文件存储 |
| Elasticsearch | 关键词搜索引擎 |
| Qdrant | 向量数据库（待embedding模型） |

### 5.2 数据流

```
用户上传文件（Word/PDF）
    ↓
后端接收 → 存储到 MinIO
    ↓
解析文件提取文本
    ↓
MySQL存元数据(title, summary, category, source, tags, file_path)
ES存完整content（全文搜索）
    ↓
用户搜索 → ES查询 → 结果展示
用户查看详情 → 从ES获取content
```

### 5.3 API 接口

| 方法 | 路径 | 说明 |
|-----|------|------|
| POST | /api/v1/knowledge/upload | 上传文件并解析 |
| GET | /api/v1/knowledge | 知识列表 |
| GET | /api/v1/knowledge/{id} | 知识详情（含ES中的content） |
| PUT | /api/v1/knowledge/{id} | 更新知识 |
| DELETE | /api/v1/knowledge/{id} | 删除知识 |
| GET | /api/v1/knowledge/search | 搜索知识 |
| POST | /api/v1/knowledge/{id}/favorite | 收藏 |
| DELETE | /api/v1/knowledge/{id}/favorite | 取消收藏 |
| GET | /api/v1/knowledge/{id}/comments | 评论列表 |
| POST | /api/v1/knowledge/{id}/comments | 添加评论 |

## 六、页面原型

- 知识管理页面：`/knowledge/manage`
- 知识详情页面：`/knowledge/detail/{id}`
- 知识搜索页面：`/knowledge/search`（复用搜索门户）
