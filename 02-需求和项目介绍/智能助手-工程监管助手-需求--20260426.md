# 工程监管助手 - 需求文档

> 创建时间：2026-04-26

## 模块定位

基于知识图谱的工程质量安全问题定位与处置推荐，为工程质量监督和安全监管提供智能辅助。

## 业务背景

工程现场检查中，检查人员需要快速获取相关技术标准、规范要求和处置建议。目前依赖人工查阅规范，效率低且容易遗漏。

## 功能需求

### 核心功能

| 功能 | 描述 | 优先级 |
|-----|------|-------|
| 问题描述输入 | 用户描述现场发现的工程质量问题 | P0 |
| 智能问答 | 基于 RAG 检索生成处置建议和技术标准 | P0 |
| 问题类型快速选择 | 预设常见问题类型，点击自动填入 | P1 |
| 相关规范推荐 | 基于问题类型推荐相关规范标准 | P1 |
| 监管记录 | 展示最近的处理记录 | P2 |

### 问题类型

| 类型 | 说明 |
|-----|------|
| 混凝土强度不足 | 混凝土强度等级不达标 |
| 钢筋配置不规范 | 钢筋数量、间距、保护层厚度不符合要求 |
| 防水工程缺陷 | 屋面、卫生间防水渗漏 |
| 脚手架安全隐患 | 脚手架搭设不符合规范 |
| 施工缝处理不当 | 施工缝位置、接茬处理不符合要求 |
| 材料以次充好 | 使用不合格建筑材料 |

### 用户流程

1. 用户输入或选择问题类型
2. 系统基于 RAG 检索相关知识
3. 返回处置建议、规范依据和相关知识卡片
4. 用户可点击卡片查看详情

## 技术方案

### 会话管理

- 使用 `qa_session` 表，`category='supervise'`
- 与执法助手(`law_general`)、问答助手(`qa`) 隔离

### RAG 流程

```
用户问题 → ES 关键词搜索 → 检索知识 → 构建 Prompt → MiniMax LLM → 生成回答
```

### 数据来源

- 知识库中的技术标准类(`tech`)知识
- 执法案例类(`case`)知识

## 验收标准

1. 前端页面能正常发送消息并接收回答
2. 回答包含处置建议和规范依据
3. 右侧面板展示相关规范和知识卡片
4. 会话历史正确保存和加载
5. 不同助手会话相互隔离
