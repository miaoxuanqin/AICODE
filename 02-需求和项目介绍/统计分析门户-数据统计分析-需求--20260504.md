## 一、项目概述

### 1.1 项目名称

统计分析门户

### 1.2 项目定位

为知识库系统提供直观的数据统计分析可视化界面，帮助用户和管理员实时掌握系统运行状态、知识资产状况和用户活动情况。

### 1.3 项目背景

用户需要一个漂亮、现代化的统计分析门户原型，能够展示：
- 知识资产的总量和增长趋势
- 索引处理状态和进度
- 知识分类分布
- 热门标签和来源统计
- 用户活跃度分析
- 快捷操作入口

## 二、功能需求

### 2.1 核心功能清单

| 序号 | 功能点 | 功能描述 | 优先级 |
|------|--------|----------|--------|
| F01 | 统计卡片展示 | 展示6个关键业务指标卡片 | P0 |
| F02 | 知识增长趋势 | 面积折线图展示知识增长趋势 | P0 |
| F03 | 分类分布分析 | 环形图/玫瑰图展示知识分类占比 | P0 |
| F04 | 索引状态监控 | 堆叠柱状图展示索引状态分布 | P0 |
| F05 | 热门标签分析 | TOP15热门标签可视化 | P1 |
| F06 | 知识来源统计 | TOP10知识来源柱状图 | P1 |
| F07 | 用户活跃度 | 热力图展示用户活跃时段 | P1 |
| F08 | 快捷操作入口 | 一键访问常用功能 | P0 |
| F09 | 最近动态 | 时间线展示最新系统动态 | P1 |
| F10 | 索引进度 | 实时显示索引任务进度 | P1 |

### 2.2 详细需求说明

#### F01: 统计卡片展示

**需求描述**：展示6个关键业务指标的统计卡片

**指标清单**：
| 指标 | 数值格式 | 趋势显示 | 进度条 |
|------|---------|---------|--------|
| 知识总数 | 12,856 | +12.5% | - |
| 本月新增 | 386 | +8.2% | - |
| 全文索引量 | 11,520 | +5.3% | 89.6% |
| 向量索引量 | 8,960 | +15.8% | 69.7% |
| 图谱节点数 | 4,528 | +22.1% | 35.2% |
| 用户总数 | 128 | +3.6% | - |

**交互要求**：
- 卡片悬停有上浮动效
- 进度条颜色与卡片主题一致
- 支持点击跳转到相关页面

#### F02: 知识增长趋势

**需求描述**：面积折线图展示知识增长趋势

**图表特性**：
- 双系列：新增知识、索引完成
- 面积填充渐变效果
- 支持时间范围切换

**切换选项**：
| 选项 | 数据范围 | 数据点 |
|------|---------|--------|
| 本周 | 周一到周日 | 7个点 |
| 本月 | 第1-4周 | 4个点 |
| 本年 | 1-12月 | 12个点 |

#### F03: 分类分布分析

**需求描述**：环形图/玫瑰图展示知识分类占比

**分类数据**：
| 分类 | 数量 | 占比 |
|------|------|------|
| 法律法规 | 4,200 | 32.7% |
| 技术标准 | 3,200 | 24.9% |
| 执法案例 | 2,400 | 18.7% |
| 政策文件 | 1,800 | 14.0% |
| 其他 | 1,256 | 9.8% |

**切换选项**：
- 环形图：适合展示比例
- 玫瑰图：适合展示多维度对比

#### F04: 索引状态监控

**需求描述**：堆叠柱状图展示各分类的索引状态分布

**状态分类**：
| 状态 | 说明 | 颜色 |
|------|------|------|
| 已完成 | 索引成功 | 绿色 |
| 进行中 | 正在处理 | 蓝色 |
| 待处理 | 等待中 | 橙色 |
| 失败 | 处理失败 | 红色 |

#### F05: 热门标签分析

**需求描述**：TOP15热门标签可视化

**切换选项**：
- 水平柱状图：适合精确对比
- 词云图：适合展示热度分布

**标签数据**：
| 排名 | 标签 | 热度 |
|------|------|------|
| 1 | 施工安全 | 156 |
| 2 | 建筑设计 | 134 |
| 3 | 消防规范 | 112 |
| ... | ... | ... |

#### F06: 知识来源统计

**需求描述**：柱状图展示TOP10知识来源

**数据维度**：来源类型 + 数量

#### F07: 用户活跃度

**需求描述**：热力日历图展示用户活跃时段

**数据维度**：
- X轴：一天内的小时（8:00-18:00）
- Y轴：一周七天
- 颜色：活跃度从低到高渐变

#### F08: 快捷操作入口

**需求描述**：提供常用功能的快捷入口

**操作清单**：
| 操作 | 图标 | 跳转路径 |
|------|------|---------|
| 上传文档 | Upload | /knowledge/manage-new?action=upload |
| 添加文本 | Edit | /knowledge/manage-new?action=text |
| 搜索知识 | Search | /knowledge/search |
| 查看图谱 | Share | /knowledge/graph |
| 智能助手 | ChatDotRound | /assistant/law |
| 同步状态 | Refresh | - (本地刷新) |

#### F09: 最近动态

**需求描述**：时间线展示最新系统动态

**动态类型**：
- upload：上传文档
- create：创建分类/知识
- update：更新文档
- graph：图谱操作
- search：检索操作

#### F10: 索引进度

**需求描述**：进度条展示索引任务进度

**进度项**：
| 名称 | 当前值 | 总数 | 进度 |
|------|--------|------|------|
| 全文索引 | 11,520 | 12,856 | 89.6% |
| 向量索引 | 8,960 | 12,856 | 69.7% |
| 知识图谱 | 4,528 | 12,856 | 35.2% |

## 三、非功能性需求

### 3.1 性能需求

| 指标 | 要求 |
|------|------|
| 页面加载时间 | < 2s |
| 图表渲染时间 | < 1s |
| 交互响应时间 | < 100ms |

### 3.2 兼容性需求

| 项目 | 要求 |
|------|------|
| 浏览器 | Chrome/Firefox/Safari/Edge 最新版本 |
| 响应式 | 支持 1920px / 1600px / 1400px / 1200px / 768px |
| 移动端 | 基本适配 |

### 3.3 视觉需求

| 项目 | 要求 |
|------|------|
| 色彩风格 | 现代扁平化，渐变色彩 |
| 主色调 | 紫色 (#667eea → #764ba2) |

| 动效 | 卡片悬停、图表交互 |
