# 知识搜索门户功能点

> 创建时间：2026-04-25

## 一、功能概述

知识搜索门户是系统的核心入口，为业务人员提供工程施工技术规范、建筑行业标准、质量安全监管要求、综合执法依据的一站式检索服务。

## 二、功能清单

### 2.1 搜索功能

| 功能点 | 描述 | 优先级 |
|-------|------|--------|
| 关键词搜索 | 支持自然语言检索 | P0 |
| 分类筛选 | 按知识分类过滤（法律法规/技术标准/执法案例/政策文件） | P0 |
| 来源筛选 | 按来源过滤（国家法规/省级法规/行业标准） | P1 |
| 日期范围 | 按发布时间范围筛选 | P1 |
| 排序方式 | 按相关度/最新/热度排序 | P1 |

### 2.2 搜索结果展示

| 功能点 | 描述 | 优先级 |
|-------|------|--------|
| 结果列表 | 展示标题、摘要、来源、发布时间、浏览量 | P0 |
| 结果分页 | 支持10/20/50/100条/页 | P0 |
| 分类标签 | 展示知识分类标签 | P0 |
| 热门标签 | 热门搜索词展示 | P1 |

### 2.3 知识详情

| 功能点 | 描述 | 优先级 |
|-------|------|--------|
| 内容展示 | 富文本内容展示 | P0 |
| 附件下载 | 支持附件下载 | P1 |
| 相关知识 | 推荐相关知识 | P1 |
| 知识信息 | 来源、发布部门、实施日期等 | P0 |

### 2.4 用户互动

| 功能点 | 描述 | 优先级 |
|-------|------|--------|
| 收藏 | 收藏知识到个人收藏夹 | P1 |
| 评论 | 对知识进行评论 | P1 |
| 点赞点踩 | 对知识进行评价 | P1 |
| 分享 | 分享知识链接 | P2 |

### 2.5 个人中心

| 功能点 | 描述 | 优先级 |
|-------|------|--------|
| 我的收藏 | 查看收藏的知识列表 | P1 |
| 搜索历史 | 查看历史搜索记录 | P1 |
| 浏览历史 | 查看浏览过的知识 | P2 |

## 三、多视角支持

### 3.1 视角切换

- 省级视角：可查看全省知识+国家法规
- 市县视角：可查看本地知识+上级授权内容

### 3.2 权限控制

- 不同视角用户搜索结果不同
- 内部知识不对外公开
- 按组织路径过滤数据
