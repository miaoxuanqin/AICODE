# 需求和项目介绍
## 项目名
项目名：行业知识图谱

## 目录目的

本目录用于描述项目背景、目标、用户场景、业务流程和范围边界，帮助学员和 AI 在开发前统一理解项目要做什么。

## 阅读顺序

1. `01-项目背景.md`
2. `02-项目目标.md`
3. `03-用户与场景.md`
4. `04-范围与边界.md`
5. `05-业务流程.md`

## 输出物

- 明确的项目目标
- 统一的业务理解
- 可用于后续架构和功能设计的需求基础

## 与其他目录的关系

- 本目录回答做什么
- `03-技术架构设计/` 回答怎么设计
- `04-功能详细设计/` 回答具体怎么实现
